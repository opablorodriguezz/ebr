# Design Context - EBR obres i reformes sl

## Design Philosophy: Minimalismo Corporativo Sofisticado

### Core Design Principles
1. **Espaço Negativo Generoso**: Layouts assimétricos com muito espaço em branco para transmitir exclusividade
2. **Tipografia Hierárquica**: Contraste forte entre títulos em serif elegante e corpo em sans-serif limpo
3. **Paleta Monocromática com Acentos**: Preto, branco, cinzas sofisticados com acentos em ouro champagne
4. **Fotografia como Protagonista**: Imagens grandes, sem molduras, sangradas nas bordas

### Color Palette
- **Background**: Branco puro (`#FFFFFF`)
- **Text Primary**: Preto profundo (`#1A1A1A`)
- **Text Secondary**: Cinza escuro (`#4A4A4A`)
- **Accent**: Ouro champagne (`#D4AF37` ou `#C9A961`)
- **Border/Divider**: Cinza muito claro (`#E8E8E8`)

### Typography
- **Display**: Playfair Display (serif elegante) para títulos principais
- **Heading**: Montserrat 600-700 para subtítulos
- **Body**: Inter 400-500 para texto corrido
- **Hierarchy**: 48px > 32px > 24px > 16px > 14px

### Visual Elements
- Linhas horizontais finas em ouro como separadores
- Números grandes em tipografia display para estatísticas
- Cartões com sombra mínima e hover elevado
- Espaçamento generoso entre seções

### Interaction Philosophy
- Transições suaves (300-500ms) para transmitir deliberação
- Hover effects sutis: aumento de opacidade ou elevação leve
- Scroll revelador: elementos aparecem conforme o usuário desce

### Animation Guidelines
- Fade-in progressivo de imagens ao scroll
- Números contadores animados
- Hover em cartões: elevação sutil (2-3px) com sombra aumentada
- Transições suaves entre seções
